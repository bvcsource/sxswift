SXSwift - How it works

What is SXSwift?

OpenStack is a free and open-source cloud computing platform. The underlying object storage that OpenStack is shipped with is known as Swift. SXSwift is an in-place replacement for Swift which allows OpenStack to communicate with SX Cluster. It translates Swift protocol into requests understandable by SX Cluster and it is fully compatible with other OpenStack components.

Install

How it works

SXSwift talks with OpenStack in Swift’s protocol. It acts as a translator between OpenStack and SX Cluster, i.e. it translates OpenStack requests into SX Cluster requests and translates SX Cluster responses into OpenStack responses. Additionally it handles two types of authentication: Keystone (OpenStack’s default authenticaton service) and native Swift’s authentication. It supports all Swift’s features making it an ideal replacement.

Features

Openstack Compatibility

SXSwift is fully compatible with other OpenStack components like Keystone or Glance. Use it as a drop-in replacement for OpenStack’s Swift.

Safe & efficient

For safety SXSwift relies on either Keystone (OpenStack’s identity server) or it’s own builtin authentication mechanism (fully compatible with Swift). SXSwift is highly efficient since it relies on SX Cluster under the hood.

Scalable

Install multiple SXSwift servers and add all of them to a round-robin to load balance the traffic among them.
